- 原文：The Founder's Playbook: Building an AI-Native Startup
- 原作者：Anthropic
- 原文链接：https://claude.com/blog/the-founders-playbook
- 中文版：[创业者手册-构建AI原生创业公司-中文.pdf](translations/founders-playbook-ai-native-startup/创业者手册-构建AI原生创业公司-中文.pdf)

### 多层电子系统的时间缩放理论

- 原文：A Time Scaling Theory for Multi-Layer Electronic Systems
- 作者：Tingbo He
- 单位：Huawei
- 来源：ChinaXiv:202605.00224v1
- 原文链接：https://chinaxiv.org/abs/202605.00224
- 中文版：[韬定律-多层电子系统的时间缩放理论-中文翻译版.pdf](translations/time-scaling-theory-multi-layer-electronic-systems/韬定律-多层电子系统的时间缩放理论-中文翻译版.pdf)
- 说明：讨论后摩尔时代的时间缩放原则，以特征时间常数 tau 作为跨器件、电路、芯片、系统的统一优化目标。

## 收录原则

- 只收录对个人学习、研究或实践有持续价值的文章、论文和资料。
- 每个条目必须注明原作者、原文标题、来源和中文译文路径。
- 翻译件保持非官方属性，不冒充原作者或原机构发布。
- 如权利人认为内容不适合公开存放，可通过 Issue 联系，我会尽快移除相关内容。
